01 — Accent system

Three-mode dual accent — the accent logic inverts per surface

Light mode

Rose
Primary — text accents, links, borders, hover states, chart lead
Lime
Secondary — fills, badges, dots, buttons, highlights, progress
✕ Lime never as text on light — 1.5:1 contrast fails WCAG

Dark mode

Lime
Primary — text, links, fills, badges, buttons, chart lead — everything
◆ Rose #D64074 appears on dark ONLY for governance constraint alerts — CAC breach, coverage violation, margin warning
