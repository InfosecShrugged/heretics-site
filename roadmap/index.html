The terminology collision.

Honest read

The /lexicon page retires terms OpptyCon is built on: MQL, Funnel, Attribution, Demand Gen, Lead, CPL, ROI, Awareness. Replacements named: APS, Acquisition Lanes, Stage Ledger, Provenance, Account, Cost per Lane Outcome, SPI delta.

Both are correct in their context: the lexicon names where the framework is going; OpptyCon uses the terms every buyer's CRM is built on. The lexicon page now carries a transitional-status callout acknowledging this. As the framework matures, the product evolves with it.
